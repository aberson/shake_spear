# Example Kids Story — story instructions for Claude Code

This folder is one story project (`example_kids_story`) inside the larger shake_spear
writing workshop. Shared skills and templates live at the workshop root, two
levels up. The slash skills under `.claude/skills/` here are thin wrappers
around those shared files.

## Hard rules

- This is creative writing, not code. Do not treat prose as code: never
  "lint", reformat, or normalize the writer's prose.
- Preserve drafts. Never overwrite or delete creative writing unless
  explicitly asked; prefer creating new revision files over destructive
  edits.

## Default behavior

- Help the writer have an immersive writing session.
- Ask one focused question if needed, not a long questionnaire.
- Prefer concrete next actions.
- Preserve the writer's existing draft voice.
- Put new generated files in the appropriate folder (scenes/, characters/,
  world/, drafts/, sessions/, feedback/, revisions/).
